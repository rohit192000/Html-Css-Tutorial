So what are html entities? As we know html uses symbols to write tags, so we can't use them in general text as they indicate that they are used to write tags. For that we use html entities. Syntax : &entity_name; or &#entity_number; We use entity number. ex- < > = and many more.

We will use html entities to show some code on the browser

Description list <dl></dl>

Description term <dt></dt>

Description definition <dd></dd>

Example

code(Descriptive term)
code tag is used for show the programs on the browser.(Description definition)
pre
pre tag is used when we want to not use the default justify of the paragraph tag.The content in the pre tag render on the browser as they were wriiten.
var
var tag is used for marking up the variables.
kbd
For marking up keyboard
samp
For marking the output.
